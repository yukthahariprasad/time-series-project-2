# time-series-project-2
Time series analysis and forecasting

## Project Goal
Explore and analyze economic time series data to understand trends, cycles, and basic forecasting.

## Dataset
Public time series data (e.g. inflation, GDP, unemployment, or fiscal indicators).

## Methods
- Time series visualization
- Stationarity checks
- Trend and seasonality analysis
- Simple forecasting models (ARIMA / moving averages)

## Status
Work in progress.
